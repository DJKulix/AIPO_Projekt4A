Notebook przedstawia implementację contrastive learning w PyTorch do uczenia reprezentacji obrazów bez użycia 
etykiet. Zawiera przygotowanie augmentacji danych, definicję encodera i projection head, trening z wykorzystaniem funkcji 
straty NT-Xent oraz ewaluację za pomocą wizualizacji t-SNE i klasyfikatora linear probe. 
Plik `przyklad.ipynb` zawiera uzupełnione bloki z kodem.
